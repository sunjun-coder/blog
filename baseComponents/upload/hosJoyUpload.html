提示

提示
<template>
        图片：
        <hosjoy-Upload v-model="fileList" :action='action' :uploadParameters='uploadParameters' style="margin:25px 0" />
        文件：
        <hosjoy-Upload showAsFileName v-model="fileNameList" :action='action' :uploadParameters='uploadParameters' style="margin:25px 0">
            <el-button size="small" type="primary">点击上传</el-button>
        </hosjoy-Upload>
    </template>

    <script>
    export default {
        name: 'hosJoyUpload',
        data () {
            return {
                action: 'http://192.168.20.248:30000/tms/files/upload',
                fileList: [
                { name: 'demo1.jpg', url: 'https://hosjoy-oss-test.oss-cn-hangzhou.aliyuncs.com/images/20191017/e340708c-d4a8-4b69-a051-6e62b78e1fd4.png' },

                ],
                uploadParameters: {
                    updateUid: '张功伟'
                },

    }
    </script>

# 属性

参数 说明 类型 可选值 默认值
action 必选参数，上传的地址 string - -
multiple 是否支持多选文件 boolean - true
uploadParameters 上传时附带的额外参数 object - -
show-file-list 是否显示已上传文件列表 boolean - false
showPreView 是否显示大图预览 boolean - true
limit 最大允许上传个数 number - -
file-size 上传图片大小限制 举例：1M, 2M,无需写单位 number - 100
disabled 是否禁用 boolean - false
accept 接受上传的文件类型（thumbnail-mode 模式下此参数无效） string '.jpg,.jpeg,.png,.gif,
.bmp,.pdf,.JPG,.JPEG,
.PBG,.GIF,.BMP,.PDF,
.xlsx,.xls,.ppt,.doc,.docx,
.rar,.zip'		 -
showAsFileName 文件名形式展示 boolean - false
